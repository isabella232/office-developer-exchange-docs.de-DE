# <a name="image-pox"></a>Bild (POX)

Das **Image-Element** enthält den Pfad eines Bilds, das zum Branding der Konfigurationsoberfläche verwendet wird.

[Response (POX)](response-pox.md)

[Bild (POX)](image-pox.md)
  
```xml
<Image/>
```

## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[Konto (POX)](account-pox.md) <br/> |Gibt die Kontoeinstellungen für den Benutzer an.  <br/> |
   
## <a name="text-value"></a>Textwert

Der Textwert stellt den Pfad eines Bilds dar, das zum Branding der Konfigurationsoberfläche verwendet wird.
