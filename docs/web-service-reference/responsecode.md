# <a name="responsecode"></a>ResponseCode

Das **ResponseCode-Element** stellt Statusinformationen zur Anforderung bereit. 
  
- [ResponseMessage](responsemessage.md) 
- [ResponseCode](responsecode.md)
  
```XML
<ResponseCode/>
```

**ResponseCodeType**
