# <a name="endtimeinminutes"></a>EndTimeInMinutes

Das **EndTimeInMinutes-Element** stellt das Ende des Arbeitstags für einen Postfachbenutzer dar.

## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[WorkingPeriod](workingperiod.md) <br/> |Enthält die Arbeitswochentage und -stunden des Postfachbenutzers.  <br/> Für dieses Element wird folgender XPath-Ausdruck verwendet:   <br/>  `/GetUserAvailabilityResponse/FreeBusyResponseArray/FreeBusyResponse/FreeBusyView/WorkingHours/WorkingPeriodArray/WorkingPeriod[i]` <br/> |
   
## <a name="text-value"></a>Textwert

Ein Textwert ist erforderlich. Der Textwert stellt das Ende des Arbeitstags um die Anzahl der Minuten dar, die seit Beginn des Tages vergangen sind. Beispiel: Endzeit: 18:00 Uhr wird durch 1080 Minuten dargestellt.
  
Der Bereich der möglichen Werte für dieses Element ist 0 bis 1440.
  
## <a name="remarks"></a>HinwBemerkungeneise

Das Schema, das dieses Element beschreibt, befindet sich im virtuellen EWS-Verzeichnis des Computers, der MicrosoftExchange Server 2007 mit installierter Clientzugriff-Serverrolle ausführt.
