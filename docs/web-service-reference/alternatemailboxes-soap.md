# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

Das **AlternateMailboxes-Element** stellt eine Auflistung alternativer Postfächer dar. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```
