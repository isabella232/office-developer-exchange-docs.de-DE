# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

Das **ArrayOfUserResponse-Element** enthält ein Array von [UserResponse (SOAP)-Elementen.](userresponse-soap.md) 
  
```XML
<ArrayOfUserResponse>
   <UserResponse/>
</ArrayOfUserResponse>
```

 **ArrayOfUserResponse**

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
