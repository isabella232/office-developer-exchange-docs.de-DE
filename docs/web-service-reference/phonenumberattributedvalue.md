# <a name="phonenumberattributedvalue"></a>PhoneNumberAttributedValue

Das **PhoneNumberAttributedValue-Element** gibt eine Instanz eines Arrays von Telefonnummern und der zugehörigen Zuschreibungen an. 
  
```XML
<PhoneNumberAttributedValue>
   <Value/>
   <Attributions/>
</PhoneNumberAttributedValue>
```

 **PhoneNumberAttributedValueType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
