# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

Das **ProtocolConnections-Element** enthält null oder mehr Protokollverbindungen. 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```
