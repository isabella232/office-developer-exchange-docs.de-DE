# <a name="externalemailaddress"></a>ExternalEmailAddress

Das **ExternalEmailAddress-Element** enthält die externe E-Mail-Adresse des Postfachs. 
  
```XML
<ExternalEmailAddress></ExternalEmailAddress>
```

**Zeichenfolge**

## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

[SearchableMailbox](searchablemailbox.md)
  
## <a name="text-value"></a>Textwert

Der Textwert des **ExternalEmailAddress-Elements** ist die externe E-Mail-Adresse des Postfachs. 
  
## <a name="remarks"></a>HinwBemerkungeneise

Dieses Element wurde in Exchange Server 2013 eingeführt.
  
Das Schema, das dieses Element beschreibt, befindet sich im virtuellen IIS-Verzeichnis, das Exchange-Webdienste hostet.
