# <a name="playonphone-um-web-service"></a>PlayOnPhone (UM-Webdienst)

Das **PlayOnPhone-Element** definiert eine Anforderung zum Wiedergeben eines Elements auf einem Telefon. 
  
[PlayOnPhone (UM-Webdienst)](playonphone-um-web-service.md)
  
```xml
<PlayOnPhone>
  <entryId>   </entryId>
  <DialString>   </DialString>
</PlayOnPhone>
```

 **complexType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[entryId (UM-Webdienst)](entryid-um-web-service.md) <br/> |Enthält den Wert, der den Bezeichner des Elements darstellt, das in einer [PlayOnPhone-Vorgangsanforderung (UM-Webdienst)](playonphone-operation-um-web-service.md) auf dem Telefon wiedergegeben werden soll.  <br/> |
|[dialString (UM-Webdienst)](dialstring-um-web-service.md) <br/> |Enthält den Wert für die zu wählende Telefonnummer.  <br/> |
   
### <a name="parent-elements"></a>Übergeordnete Elemente

Keine
  
## <a name="text-value"></a>Textwert

Keine.
