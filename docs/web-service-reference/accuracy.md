# <a name="accuracy"></a>Genauigkeit

Das **Accuracy-Element** gibt die Genauigkeit des Breiten- und Längengrads der zugeordneten Postadresse an. 
  
```XML
<Accuracy></Accuracy>
```

 **xs:double**
