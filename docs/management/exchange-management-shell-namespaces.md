# <a name="exchange-management-shell-namespaces"></a>Exchange-Verwaltungsshell Namespaces

Hier finden Sie Informationen zu den Namespaces für die Exchange-Verwaltungsshell in Exchange.
  
**Gilt für:** Exchange Online | Exchange Server 2013 | Office 365
  
Exchange-Verwaltungsshell Anwendungen für Exchange Online, Exchange Online als Teil Office 365 oder eine Version von Exchange, die mit Exchange 2013 beginnt, interagieren mit der Windows PowerShell Umgebung über die Typen, die vom **System. Management. Automation-** Namespace verfügbar gemacht werden. Die Exchange-Verwaltungsshell-Cmdlets verwenden Typen aus einer Reihe von Exchange-Bibliotheken. Dieser Abschnitt enthält Referenzinformationen zu den Typen und Schnittstellen aus den Exchange-Bibliotheken, die von Exchange-Verwaltungsshell-Cmdlets verwendet werden.
