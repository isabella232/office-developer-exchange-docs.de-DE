# <a name="exchange-management-shell-namespaces"></a>Namespaces für Exchange-Verwaltungsshell

Hier finden Sie Informationen zu den Namespaces für die Exchange-Verwaltungsshell in Exchange.
  
**Gilt für:** Exchange Online | Exchange Server 2013 | Office 365
  
Exchange Verwaltungsshell-Anwendungen für Exchange Online, Exchange Online als Teil Office 365 oder eine Version von Exchange ab Exchange 2013 interagieren mit der Windows PowerShell Umgebung über die Typen, die von der **System.Management.Automation-Namespace.** Die Cmdlets der Exchange Verwaltungsshell verwenden Typen aus einer Reihe von Exchange Bibliotheken. Dieser Abschnitt enthält Referenzinformationen zu den Typen und Schnittstellen aus den Exchange Bibliotheken, die von Exchange Verwaltungsshell-Cmdlets verwendet werden.
